| Surface | What it is for |
| --- | --- |
| Dashboard | Summary and recent activity. |
| Checkouts | Create, list, inspect, and deactivate checkouts. |
| Payments | Review payment status and transaction information. |
| Developers | Manage API keys, your webhook endpoint, and delivery history. |
| Settings | Account, store profile, logo, payout wallet, and store status. |

The admin console is a separate, internal-only surface. It is not part of this merchant guide and is not a supported integration API.
