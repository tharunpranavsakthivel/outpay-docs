| Base mainnet (chain ID `8453`) | Native USDC | `0x833589fCD6EDb6E08f4c7C32D4f71b54bdA02913` | Available in the seeded catalog |

Base is an EVM network. The payment matcher compares network, token contract, recipient address, and amount before it records a match.

**Not implemented:** Polygon, Ethereum mainnet, Solana, ETH, USDT, multi-chain routing, custodial balances, and a payment-router smart contract.
